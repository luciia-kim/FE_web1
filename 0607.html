web1-16. 원시웹

web 과 internet은 다른 것. web은 internet안에 존재하는 서비스.
1960년 internet의 등장, 30년 후 1990년 web이 등장함.

  • Internet 등장(1960)
    핵공격에도 강인한 통신 시스템의 필요 - 하나의 메인으로 이루어진 것이 아닌, 분산된 형태의 통신시스템이기 때문에 핵공격에도 안전했음. Internet은 web 출현 이전까지는 정부, 군대, 대학 등 큰 조직에서만 사용하는 것이었음.
  • web의 등장(1990)
    swiss 제네바 물리학 연구소 cern의 버너스리 연구원팀이 만듦. 팀 버너스리가 1990년 10월 최초의 웹페이지 편집기 제작, 11월 World Wide Web제작, 12월 24일 info.cern.ch 라는 주소의 최초의 웹페이지를 만듦.



web1-17. 서버와 클라이언트

Web Browser, Web Server 두 대의 컴퓨터.
웹서버 컴퓨터에는 http//info.cern.ch 라는 주소를 갖고 있으며, 하드디스크가 있고, index.html이라는 파일을 갖고 있음.
웹 브라우저 컴퓨터에서는 http://info.cern.ch/index.html 이라는 주소를 입력했을 때, 웹서버 컴퓨터는 index.html 파일의 전기적 신호를 보내주게 됨.
Web Browser 컴퓨터는 Web Server 컴퓨터에 정보를 요청(request)하고, Web Server 컴퓨터는 Web Browser 컴퓨터에 응답(response)한다.
이 관계를 서버와 클라이언트로 정의하는데, 요청하는 컴퓨터가 client, 응답하는 컴퓨터를 server 컴퓨터로 부른다.

Web Browser = (Web/Game/Chatting...) Client / Web Server = (Web/Game/Chatting...) Server


web1-18. 웹호스팅-github page

github: 세계적으로 유명한 웹 호스팅 업체, 무료로 웹서버 제공해주는 곳.
전세계 수많은 오픈소스들이 여기서 만들어짐
github.com 에서 create repository(제작한 소스코드 저장소) 클릭해서 저장소 이름, 상태는 public, initialize with은 readme 선택 후 생성.
생성된 저장소에서 파일 업로드를 통해 제작한 소스코드 파일을 저장해 둘 수 있음. commit changes에 무슨 파일인지 메모하고 저장하면 됨.
